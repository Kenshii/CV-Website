Software

Mobile IRL – Android App (Java) – 2017
This is a work-in-progress open-source alternative to the My Vodafone Ireland app.
It communicates with the My Vodafone mobile API via the OkHttp3 and Gson libraries.
A long-term goal is to allow a user to log in from other operators via the same app.
Home Monitoring System – Android App (Java) – 2017
This is a group project that is being worked on throughout the first semester of third year.
It runs as a service and listens for voice commands constantly. These commands can perform several actions such as taking pictures with the phone’s camera, toggling strips of LED’s on an Arduino and opening a memory-based mini-game in the app.
The app is controllable via both voice commands and a responsive menu.
Reverse Phone Lookup Tool (Python) – 2014, 2017
This is a Python script that lets you input an Eircom phone number and obtain the name and address which correspond with that phone number. It matches it against a local database.
To create it, I needed to write a separate script that scraped the entirety of Eircom’s phonebook database. This script involves JSON deserialization, HTML parsing, regular expressions and an ORM.
It was rewritten from scratch in 2017, hence why I chose to show two years above.
CIT Dentist Management App (JavaFX) – 2016 – Result: 90%
This is a JavaFX application that stores dentistry information in a locally hosted Apache Derby database.
It makes use of the MVC and singleton design patterns, and lets you create, edit and delete dentists and their patients, patients’ invoices, payments and procedures and generate reports detailing them.
500px Bot (Python) – 2015
This bot essentially follows users as they post trending images on 500px, and unfollows those users if they unfollow you later or if they do not follow you back within 48 hours.
It is intended to be ran as a daily cronjob, and makes use of the requests and BeautifulSoup libraries.
