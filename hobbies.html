Uncovering Website Vulnerabilities

I have found and reported multiple vulnerabilities in websites over the last few years.

Dunnes Stores

I noticed the secret key used in Dunnes' Gift Card payment system was being leaked in its gift-card related pages.

This secret key would have given people the ability to purchase €100 gift cards for any arbitrary amount of money.

E-xamit

There were multiple SQL injection vulnerabilities on E-xamit's web pages.

One such affected page was tutorial.php

Hummingbird

I found a persistent XSS vulnerability on user profile pages, as demonstrated here.

I also reported accidentally leaked admin access tokens when the website was made open-source.

Groupees

Payment methods were each associated with numbers on Groupees' website.

Paying with payment method 0 enabled users to obtain bundles without paying for them.
